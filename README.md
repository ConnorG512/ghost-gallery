# **Ghost Gallery**

A game created with C++ and Raylib. Shoot the ghosts and get the highest score possible! 

You can find the game download on [itch.io](https://gronallacon.itch.io/ghost-gallery).

## **Controls:**

**Fire:** Left mouse Button

**Start Game:** Space

**Exit:** ESC

**Volume up:** P

**Volume down:** O
